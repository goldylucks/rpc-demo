# RPC Demo

# develop
```bash
$ git clone git@github.com:goldylucks/rpc-demo.git
$ cd rpc-demo
$ yarn
$ yarn start # starts express dev server on localhost:8000 
# another terminal
$ yarn dev:wds # starts webpack-dev-server on localhost:7000 
```

open your favorite browser and hack away :)
